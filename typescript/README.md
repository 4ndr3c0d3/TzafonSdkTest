# TypeScript — Tzafon Automations

Install
- In this folder: `npm install`
- Ensure `export TZAFON_API_KEY=sk_your_api_key_here`

Automations (immediate execution)
- `npm run wiki`
- `npm run nytimes`
- `npm run airbnb`
- `npm run github`
- `npm run reddit`

Concurrency
- `npm run concurrent:10` — launches 10 browsers simultaneously
- `npm run concurrent:50` — launches 50 browsers simultaneously

Results
- Screenshots saved under `typescript/results/…` with timestamped filenames.
