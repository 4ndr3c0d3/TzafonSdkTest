# Python — Tzafon Automations

Install
- `pip install -r requirements.txt`
- Ensure `export TZAFON_API_KEY=sk_your_api_key_here`

Automations (immediate execution)
- `python wikipedia.py`
- `python nytimes.py`
- `python airbnb.py`
- `python github.py`
- `python reddit.py`

Concurrency
- `python concurrent_10.py` — launches 10 browsers simultaneously
- `python concurrent_50.py` — launches 50 browsers simultaneously

Results
- Screenshots saved under `python/results/…` with timestamped filenames.
